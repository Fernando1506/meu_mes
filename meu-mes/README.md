# Meu Mês — versão comentada

Esta versão mantém a estrutura organizada em pastas e adiciona comentários explicativos nos principais arquivos do projeto.

## Arquivos principais

- `index.html`: estrutura HTML e links para CSS/JS.
- `css/`: estilos separados por responsabilidade.
- `js/`: lógica separada por responsabilidade.
- `MANUAL_DE_MANUTENCAO.md`: guia de onde mexer para alterar cada parte.

## Como abrir

Abra `index.html` no navegador.

## Observação

A ordem dos arquivos CSS e JS no `index.html` deve ser preservada.
